Infos über Hyriki

Die Republik Hyriki (kroatisch Republika Hyriki) ist eine direktorial-demokratische Republik im Südosten von Absurdistan. Sie liegt südöstlich von Schuckland und östlich von Tembolien. Sie ist seit Mai 2023 Mitglied der CONAS.

Wahlspruch „Ponosni na našu povijest, nada naše budućnosti“ (Stolz auf unsere Geschichte, Hoffnung auf unsere Zukunft)
Amtssprachen Kroatisch, Deutsch
Hauptstadt Sveti Centar (Osten Hyriki)
Staatsform präsidentielle Republik
Fläche 22.403 km²
Einwohner:innen 5.161.290
